# 基金投资分析工具 - 日常常用命令

## 命令结构

```
主程序: python scripts/main_cli.py
独立脚本:
  - python scripts/daily.py           # 日常操作
  - python scripts/knowledge.py       # 知识库
  - python scripts/analysis_tools.py  # 高级分析
```

---

## 每日必用（只需一个命令）

### 晨报（每日必看）

```bash
python scripts/daily.py --morning
python scripts/daily.py --morning --devil  # 含反面观点
python scripts/daily.py --morning --quiet  # 安静模式
```

### 定投（记录投资）

```bash
python scripts/daily.py --invest
```

### 交易信号（看买卖建议）

```bash
python scripts/main_cli.py --action signal --fund 110011
python scripts/main_cli.py --action signal --fund 110011 --devil  # 含反面观点
```

---

## 每周必用

### 假设分析（如果当时定投会怎样）

```bash
python scripts/main_cli.py --action what-if --fund 110011
python scripts/main_cli.py --action what-if --fund 110011 --amount 1000 --years 5
```

### 投资笔记（记录思考）

```bash
# 添加笔记
python scripts/main_cli.py --action note --book "聪明的投资者" --text "安全边际是投资的核心"

# 查看笔记
python scripts/main_cli.py --action note
```

---

## 偶尔使用

### 月底复盘（含随机提问）

```bash
python scripts/daily.py --review
```

### 列出基金

```bash
python scripts/main_cli.py --action list
```

### 配置管理

```bash
python scripts/main_cli.py --action config
```

---

## 高级分析（按需使用）

```bash
python scripts/analysis_tools.py --help
python scripts/analysis_tools.py analyze --fund 110011    # 深度分析
python scripts/analysis_tools.py compare --fund 110011    # 基金对比
python scripts/analysis_tools.py backtest --fund 110011   # 回测
python scripts/analysis_tools.py simulate --fund 110011   # 定投模拟
python scripts/analysis_tools.py chart --fund 110011      # 图表
python scripts/analysis_tools.py report --fund 110011     # 报告
python scripts/analysis_tools.py timeline                 # 时间线
```

---

## 知识库（按需使用）

```bash
python scripts/knowledge.py --books      # 书籍推荐
python scripts/knowledge.py --quotes     # 投资金句
python scripts/knowledge.py --check-risk --text "保本保息"  # 风险检测
python scripts/knowledge.py --note       # 笔记管理
```

## 快速参考

| 场景 | 命令 |
|------|------|
| 每日晨报 | `python daily.py --morning` |
| 晨报+反面观点 | `python daily.py --morning --devil` |
| 定投记录 | `python daily.py --invest` |
| 看买卖信号 | `--action signal --fund CODE` |
| 信号+反面观点 | `--action signal --fund CODE --devil` |
| 假设分析 | `--action what-if --fund CODE` |
| 记笔记 | `--action note --book "书名" --text "内容"` |
| 月底复盘 | `python daily.py --review` |
| 深度分析 | `python analysis_tools.py analyze --fund CODE` |

---

**提示**：
- 将 `110011` 替换为你想分析的基金代码
- `--quiet` 参数只在有重要信号时输出
- `--devil` 参数显示反面观点
- 高级分析请运行 `python scripts/analysis_tools.py --help`
- 知识库请运行 `python scripts/knowledge.py --help`
