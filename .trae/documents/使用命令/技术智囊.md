# 基金投资分析工具 - 技术智囊

## 项目概述

这是一个专为500元入门投资者设计的基金投资分析工具，帮助你建立科学的投资决策体系。

---

## 一、项目架构

### 1.1 命令结构

```
主程序: python scripts/main_cli.py
独立脚本:
  - python scripts/daily.py           # 日常操作（晨报、定投、复盘）
  - python scripts/knowledge.py       # 知识库（书籍、金句、风险检测、笔记）
  - python scripts/analysis_tools.py  # 高级分析（深度分析、对比、回测、图表）
```

### 1.2 核心模块

| 模块 | 文件 | 功能 |
|------|------|------|
| 数据获取 | fund_data_fetcher_v2.py | 多数据源降级（东方财富、新浪、腾讯、AKShare） |
| 数据分析 | fund_analyzer_v2.py | 技术指标、风险指标、风格归因 |
| 交易系统 | trading_system.py | 五模块信号系统（趋势、买点、波动、风控、止盈） |
| 数据库 | fund_database.py | SQLite存储、投资日志、时间线 |
| 通知 | fund_notifier.py | 钉钉、微信、邮件通知 |
| 配置 | config_manager.py | 分级配置（default + user） |
| 知识库 | knowledge_base.py | 书籍、金句、风险检测、笔记 |

## 二、核心功能详解

### 2.1 五模块交易系统

#### 趋势模块
- **指标**：EMA均线（5日、20日、60日）
- **判断**：
  - 短期均线 > 长期均线 → 趋势向上
  - 短期均线 < 长期均线 → 趋势向下
- **命令**：
```bash
python scripts/main_cli.py --action signal --fund 110011
```

#### 买点模块
- **指标**：RSI、MACD
- **判断**：
  - RSI < 30 → 超卖，买入机会
  - RSI > 70 → 超买，卖出信号
  - MACD金叉 → 买入信号
  - MACD死叉 → 卖出信号
- **命令**：
```bash
python scripts/main_cli.py --action signal --fund 110011
```

#### 波动模块
- **指标**：布林带、波动率
- **判断**：
  - 波动率高 → 风险大，减少仓位
  - 波动率低 → 风险小，可适当加仓
- **命令**：
```bash
python scripts/main_cli.py --action signal --fund 110011
```

#### 风控模块
- **指标**：最大回撤
- **判断**：
  - 回撤超过阈值 → 暂停定投
  - 回撤在可控范围 → 继续定投
- **命令**：
```bash
python scripts/main_cli.py --action signal --fund 110011
```

#### 止盈模块
- **指标**：收益率目标
- **判断**：
  - 达到目标收益 → 考虑止盈
  - 未达到目标 → 继续持有
- **命令**：
```bash
python scripts/main_cli.py --action signal --fund 110011
```

---

### 2.2 魔鬼代言人模式

**功能**：显示反面观点，帮助避免冲动决策

**使用场景**：
- 当你想买入时，提醒你风险
- 当你想卖出时，提醒你机会
- 帮助你理性思考

**命令**：
```bash
# 在signal命令中使用
python scripts/main_cli.py --action signal --fund 110011 --devil

# 在daily.py --morning中使用
python scripts/daily.py --morning --devil
```

**输出示例**：
```
[魔鬼代言人] 反面观点

易方达中小盘混合 (110011):
  - 该基金近30天下跌了11.7%。
    建议：在市场下跌时卖出往往卖在低点，建议冷静思考。

招商中证白酒指数 (161725):
  - 过去一年收益率超过30%，注意均值回归风险
```

---

### 2.3 假设分析（what-if）

**功能**：假设过去开始定投，现在会怎样

**使用场景**：
- 评估定投策略的效果
- 对比不同金额的定投结果
- 了解长期定投的收益

**命令**：
```bash
# 默认假设（500元/月，3年）
python scripts/main_cli.py --action what-if --fund 110011

# 自定义金额和年数
python scripts/main_cli.py --action what-if --fund 110011 --amount 1000 --years 5
```

**输出内容**：
- 总投入金额
- 最终市值
- 总收益率
- 年化收益率
- 每月定投明细

---

### 2.4 随机提问功能

**功能**：月底复盘时，从笔记或金句库中随机抽取一条，问你"这个月你有做到吗？"

**使用场景**：
- 促进反思和成长
- 复盘更有深度

**命令**：


**输出示例**：

```
  问题：这个月你有做到吗？
```

## 六、技术指标

### 6.1 趋势指标

| 指标 | 说明 | 用途 |
|------|------|------|
| EMA | 指数移动平均线 | 判断趋势方向 |
| MA | 简单移动平均线 | 判断支撑/阻力 |

### 6.2 动量指标

| 指标 | 说明 | 用途 |
|------|------|------|
| RSI | 相对强弱指标 | 判断超买/超卖 |
| MACD | 指数平滑异同移动平均线 | 判断买卖信号 |

### 6.3 波动指标

| 指标 | 说明 | 用途 |
|------|------|------|
| 布林带 | 价格通道 | 判断波动范围 |
| 波动率 | 价格波动幅度 | 判断风险大小 |

### 6.4 风险指标

| 指标 | 说明 | 用途 |
|------|------|------|
| 最大回撤 | 最大亏损幅度 | 判断风险承受能力 |
| 夏普比率 | 风险调整后收益 | 判断投资效率 |
| 信息比率 | 超额收益/跟踪误差 | 判断主动管理能力 |

## 七、数据库

### 7.1 数据库结构

```sql
-- 投资日志表
CREATE TABLE journal (
    id INTEGER PRIMARY KEY,
    fund_code TEXT,
    action TEXT,  -- buy/sell/hold
    amount REAL,
    nav REAL,
    reason TEXT,
    emotion TEXT,
    reflection TEXT,
    created_at TIMESTAMP
);

-- 定投记录表
CREATE TABLE invest_records (
    id INTEGER PRIMARY KEY,
    fund_code TEXT,
    amount REAL,
    nav REAL,
    created_at TIMESTAMP
);
```
