# 基金投资分析工具 - 完整命令手册

## 命令结构

```
主程序: python scripts/main_cli.py
独立脚本:
  - python scripts/daily.py           # 日常操作（晨报、定投、复盘）
  - python scripts/knowledge.py       # 知识库（书籍、金句、风险检测、笔记）
  - python scripts/analysis_tools.py  # 高级分析（深度分析、对比、回测、图表）
```

---

## 一、核心命令（main_cli.py）

### 1.1 signal - 交易信号

```bash
# 基本信号分析
python scripts/main_cli.py --action signal --fund 110011

# 含反面观点（魔鬼代言人模式）
python scripts/main_cli.py --action signal --fund 110011 --devil

# 指定分析天数
python scripts/main_cli.py --action signal --fund 110011 --days 180
```

**输出内容**：
- 趋势模块：EMA均线判断
- 波动模块：布林带+波动率
- 买点模块：RSI+MACD
- 风控模块：回撤控制
- 止盈模块：收益目标
- 综合建议：仓位比例

**--devil参数**：
- 显示反面观点
- 提醒常见错误
- 帮助避免冲动决策

---

### 1.2 what-if - 假设分析

```bash
# 默认假设（500元/月，3年）
python scripts/main_cli.py --action what-if --fund 110011

# 自定义金额和年数
python scripts/main_cli.py --action what-if --fund 110011 --amount 1000 --years 5
```

**输出内容**：
- 假设3年前开始定投的结果
- 总投入、最终市值、总收益
- 年化收益率
- 每月定投明细

---

### 1.3 note - 投资笔记

```bash
# 添加笔记

python scripts/main_cli.py --action note --text "安全边际"
```

## 二、日常操作（daily.py）

### 2.1 morning - 晨报（每日必用）

```bash
# 正常晨报（含信号）
python scripts/daily.py --morning

# 含反面观点（魔鬼代言人模式）
python scripts/daily.py --morning --devil

# 安静模式（只在有重要信号时输出）
python scripts/daily.py --morning --quiet

# 详细模式（显示完整分析）
python scripts/daily.py --morning --verbose
```

**输出内容**：
- 今日日期
- 每只基金的趋势和波动状态
- 操作建议
- 魔鬼代言人反面观点（使用--devil时）

---

### 2.2 invest - 定投

```bash
# 记录定投
python scripts/daily.py --invest

# 详细模式
python scripts/daily.py --invest --verbose
```

**输出内容**：
- 定投记录
- 买入金额和净值
- 总投入

---

### 2.3 review - 复盘

```bash
# 月底复盘
python scripts/daily.py --review

# 详细复盘
python scripts/daily.py --review --verbose
```

**输出内容**：
- 本月定投汇总
- 当前市值
- 总收益率
- 反思提示
- 随机回顾（从笔记或金句库中随机抽取）

## 四、高级分析（analysis_tools.py）

### 4.1 analyze - 深度分析

```bash
python scripts/analysis_tools.py analyze --fund 110011
python scripts/analysis_tools.py analyze --fund 110011 --days 180
```

**输出内容**：
- 基金名称和净值
- 总收益率、年化收益率
- 最大回撤、波动率
- 夏普比率

---

### 4.2 compare - 基金对比

```bash
# 与沪深300对比
python scripts/analysis_tools.py compare --fund 110011

# 指定基准
python scripts/analysis_tools.py compare --fund 110011 --benchmark 110030

# 指定天数
python scripts/analysis_tools.py compare --fund 110011 --days 365
```

**输出内容**：
- 基金收益率
- 基准收益率
- 超额收益
- 信息比率

---

### 4.3 backtest - 回测

```bash
# 基本回测
python scripts/analysis_tools.py backtest --fund 110011

# 自定义均线参数
python scripts/analysis_tools.py backtest --fund 110011 --short-ma 5 --long-ma 20
```

**输出内容**：
- 回测期间
- 总收益率
- 年化收益率
- 最大回撤
- 交易次数

**输出内容**：

- 总投入

- 年化收益率

## 九、常见问题

### Q1: 如何添加新基金？

编辑 `scripts/user_config.yaml`，添加：

```yaml
funds:
  - code: "基金代码"
    name: "基金名称"
    monthly_invest: 定投金额
```

### Q2: 如何配置钉钉通知？

编辑 `scripts/user_config.yaml`，添加：

```yaml
notification:
  method: "dingtalk"
  dingtalk:
    webhook: "your_webhook_here"
    secret: "your_secret_here"
```

### Q3: 数据不足怎么办？

增加 `--days` 参数：

```bash
python scripts/main_cli.py --action signal --fund 110011 --days 365
```

```bash
python scripts/main_cli.py --help

python scripts/analysis_tools.py --help
```
