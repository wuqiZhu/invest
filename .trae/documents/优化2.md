2. 技术指标计算性能 – 向量化优化
你目前大概率是用循环计算RSI、MACD。Pandas有现成的向量化实现，可以提速10倍以上：

python
def compute_rsi(series, period=14):
    delta = series.diff()
    gain = (delta.where(delta > 0, 0)).rolling(window=period).mean()
    loss = (-delta.where(delta < 0, 0)).rolling(window=period).mean()
    rs = gain / loss
    rsi = 100 - (100 / (1 + rs))
    return rsi

⚠️ 需要警惕的“伪信号”和常见误区
1. 技术指标在基金上的失效风险
基金的净值不是连续交易价格，尤其是场外基金，每天只有一个净值。RSI、MACD、布林带等指标在设计上是为了日内高频数据，用在日频净值上会严重滞后。
建议：明确在文档中标注“技术指标仅供学习参考，不可用于高频择时”。

2. 回测结果过度美化 – 过拟合陷阱
你写的双均线回测（短期MA上穿长期MA买入）在历史数据上可能很漂亮，但实盘会不断被打脸。原因：

交易成本：场外基金申购赎回费率（0.15%～1.5%）会吃掉大部分收益。

资金效率：信号出现时可能已经收盘，无法以信号价格成交。
