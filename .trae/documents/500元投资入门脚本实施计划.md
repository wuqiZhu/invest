# 500元投资入门脚本实施计划

## 一、项目概述

基于投资方案文档分析，针对500元资金的投资入门需求，制定一个循序渐进的投资脚本实施计划。本计划采用**路线A+B的轻量组合**，从最简单的实时监控开始，逐步深入到数据分析，最终实现自动化监控。

### 核心目标
1. **快速上手**：5分钟内部署实时监控，直观感受基金净值变化
2. **技能复用**：利用现有Python和前端经验，降低学习成本
3. **成本控制**：全部使用免费工具和开源项目
4. **循序渐进**：从简单到复杂，逐步建立投资分析能力

### 4.3 基金分析模块

#### 基础版 (fund_analyzer.py)
- 计算收益率指标（总收益率、年化收益率）
- 计算风险指标（最大回撤、波动率、夏普比率）
- 定投模拟
- 生成分析报告

#### 增强版 (fund_analyzer_v2.py)
- **技术指标计算**：
  - 均线：MA5、MA10、MA20、MA60
  - 20日均线偏离（超买超卖信号）
  - RSI指标（相对强弱指数）
  - 布林带（上轨、中轨、下轨）
  - MACD（DIF、DEA、MACD值）
- **信号判断**：
  - 均线偏离 > 5%：超买信号
  - 均线偏离 < -5%：超卖信号
  - RSI > 70：超买信号
  - RSI < 30：超卖信号

#### 核心代码

```python
from fund_analyzer_v2 import FundAnalyzerV2

# 创建增强版分析器
analyzer = FundAnalyzerV2(risk_free_rate=0.03)

# 分析基金（包含技术指标）
analysis = analyzer.analyze_fund(fund_data, '110011')

# 获取技术指标
indicators = analyzer.calculate_technical_indicators(fund_data['单位净值'])
print(f"MA20: {indicators['MA20']}")
print(f"RSI: {indicators['RSI']}")
print(f"20日均线偏离: {indicators['20日均线偏离']}%")
print(f"均线信号: {indicators['均线信号']}")
```

#### 技术指标说明

| 指标 | 说明 | 应用场景 |
|------|------|----------|
| MA5/10/20/60 | 移动平均线 | 判断趋势方向 |
| 20日均线偏离 | 当前价格偏离均线的程度 | 超买超卖判断 |
| RSI | 相对强弱指数（0-100） | 超买超卖判断 |
| 布林带 | 价格波动区间 | 支撑压力判断 |
| MACD | 指数平滑异同移动平均线 | 趋势转折判断 |

```sql
-- 基金基本信息表
CREATE TABLE fund_info (
    fund_code TEXT PRIMARY KEY,

-- 基金净值表

    id INTEGER PRIMARY KEY AUTOINCREMENT,

    update_time TEXT

-- 分析结果表
CREATE TABLE fund_analysis (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    fund_code TEXT,

    annualized_return REAL,
    max_drawdown REAL,
    volatility REAL,
    sharpe_ratio REAL,
    update_time TEXT
);

-- 投资记录表
```

## 七、分析指标说明

### 7.1 收益指标

| 指标 | 说明 | 计算公式 |
|------|------|----------|
| 总收益率 | 投资期间的总收益百分比 | (最新净值/初始净值 - 1) × 100% |
| 年化收益率 | 将总收益换算成年度收益率 | (最新净值/初始净值)^(365/天数) - 1 |

### 7.2 风险指标

| 指标 | 说明 | 计算公式 |
|------|------|----------|
| 最大回撤 | 从最高点到最低点的最大跌幅 | min((净值-最高净值)/最高净值) |
| 波动率 | 收益率的标准差（年化） | 日收益率标准差 × √252 |
| 夏普比率 | 风险调整后收益 | (年化收益率 - 无风险利率) / 波动率 |

### 7.3 技术指标（新增）

| 指标 | 说明 | 应用场景 |
|------|------|----------|
| MA5/10/20/60 | 移动平均线 | 判断趋势方向 |
| 20日均线偏离 | 当前价格偏离均线的程度 | 超买超卖判断 |
| RSI | 相对强弱指数（0-100） | 超买超卖判断 |
| 布林带 | 价格波动区间 | 支撑压力判断 |
| MACD | 指数平滑异同移动平均线 | 趋势转折判断 |

### 7.4 信号判断规则

| 指标 | 超买信号 | 超卖信号 | 正常区间 |
|------|----------|----------|----------|
| 20日均线偏离 | > 5% | < -5% | -5% ~ 5% |
| RSI | > 70 | < 30 | 30 ~ 70 |
| 布林带位置 | > 80% | < 20% | 20% ~ 80% |

### 7.5 风险等级划分

| 最大回撤 | 风险等级 |
|----------|----------|
| < 10% | 低风险 |
| 10% - 20% | 中风险 |
| > 20% | 高风险 |
